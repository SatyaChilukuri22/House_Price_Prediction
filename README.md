# House Price Prediction

## Project Overview

This project predicts house prices using Machine Learning techniques. The dataset contains information about house area, bedrooms, bathrooms, stories, parking, furnishing status, and other property features.

## Tools Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

## Models

* Linear Regression
* Random Forest Regressor

## Evaluation Metrics

* MAE (Mean Absolute Error)
* RMSE (Root Mean Squared Error)
* R² Score

## Visualizations

* House Price Distribution
* Correlation Heatmap
* Actual vs Predicted Prices
